Examples of JavaScript incude:

1) Example of how to Change HTML Content: document.getElementById("demo").innerHTML = "Hello JavaScript";

2) Example of how to change HTML styles (CSS): document.getElementById("demo").style.fontSize = "25px";

or

document.getElementById('demo').style.fontSize = '25px';

3) Example of how you can Hide HTML elements: document.getElementById("demo").style.display = "none";

or

document.getElementById('demo').style.display = 'none';

4) Example of how to you can show HTML elements: document.getElementById("demo").style.display = "block";

or

document.getElementById('demo').style.display = 'block';
